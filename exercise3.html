Exercise 3: 'this' with bind() Method

This exercise demonstrates how the bind() method preserves the this context when passing object methods as event handlers.

The CounterApp object has increment and decrement methods that use this.count. Without bind(), this would refer to the button element instead of the CounterApp object.

incBtn.addEventListener("click", CounterApp.increment.bind(CounterApp));
decBtn.addEventListener("click", CounterApp.decrement.bind(CounterApp));
