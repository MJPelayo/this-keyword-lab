Exercise 2: 'this' with Data Attributes

This exercise demonstrates how the this keyword can access data attributes from HTML elements.

Each color swatch has a data-color attribute containing its color value. When clicked, the applyColor() function uses this to access the specific swatch's data attribute and apply the color.

function applyColor() {
  const color = this.dataset.color;
  this.style.backgroundColor = color;
}
