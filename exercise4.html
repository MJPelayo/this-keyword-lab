Exercise 4: 'this' in Arrow Functions

This exercise demonstrates how arrow functions handle the this keyword differently from regular functions.

Arrow functions do not have their own this context. Instead, they inherit this from their surrounding lexical scope.

box.addEventListener("click", (event) => {
  // "this" does not refer to the clicked element in arrow functions
  event.currentTarget.classList.add("active");
});

Regular Function

In regular functions:
- this refers to the element that triggered the event

function(event) {
  this.classList.add("active");
}

Arrow Function

In arrow functions:
- this inherits from lexical scope
- Use event.currentTarget instead

(event) => {
  event.currentTarget.classList.add("active");
}
